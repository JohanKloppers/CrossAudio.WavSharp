# CrossAudio.WavSharp

[![NuGet](https://img.shields.io/nuget/v/CrossAudio.WavSharp.svg)](https://www.nuget.org/packages/CrossAudio.WavSharp/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![.NET](https://img.shields.io/badge/.NET-8.0%2B-blue.svg)](https://dotnet.microsoft.com/)

A high-performance, pure C# WAV audio codec library for .NET. This library provides comprehensive WAV file reading and writing capabilities with support for metadata, cue points, and sampler information.

## Features

- **Complete WAV Support**: Read and write WAV files with all standard formats (8-bit, 16-bit, 24-bit, 32-bit PCM)
- **Metadata Handling**: Full support for LIST INFO chunks, sampler information, and cue points
- **Streaming API**: Efficient streaming operations with seek, rewind, and chunked reading
- **Cross-Platform**: Pure .NET implementation, works on Windows, Linux, and macOS
- **High Performance**: Optimized for speed and memory efficiency
- **Comprehensive Testing**: Extensive test suite covering all functionality

## Installation

Install via NuGet:

```bash
dotnet add package CrossAudio.WavSharp
```

Or via Package Manager:

```powershell
Install-Package CrossAudio.WavSharp
```

## Quick Start

### Reading a WAV File

```csharp
using CrossAudio.WavSharp;

// Read a WAV file
var reader = new WavReader();
var wavFile = reader.Read("path/to/audio.wav");

// Access basic properties
Console.WriteLine($"Channels: {wavFile.NumChannels}");
Console.WriteLine($"Sample Rate: {wavFile.SampleRate} Hz");
Console.WriteLine($"Bits Per Sample: {wavFile.BitsPerSample}");
Console.WriteLine($"Duration: {reader.Duration()}");

// Access metadata if available
if (wavFile.Metadata != null)
{
    Console.WriteLine($"Title: {wavFile.Metadata.Title}");
    Console.WriteLine($"Artist: {wavFile.Metadata.Artist}");
}
```

### Writing a WAV File

```csharp
using CrossAudio.WavSharp;

// Create a WAV file from raw PCM data
var writer = new WavWriter();
var metadata = new Metadata
{
    Title = "My Audio",
    Artist = "CrossAudio",
    Comments = "Generated by CrossAudio.WavSharp"
};

writer.Write("output.wav", sampleRate: 44100, bitsPerSample: 16, channels: 2, pcmData, metadata);
```

### Streaming Operations

```csharp
using CrossAudio.WavSharp;

// Open for streaming operations
var reader = new WavReader();
reader.Open("audio.wav");

// Read PCM data in chunks
var buffer = new byte[4096];
int bytesRead;
while ((bytesRead = reader.PCMBuffer(buffer, 0, buffer.Length)) > 0)
{
    // Process audio data...
}

// Seek to a specific position
reader.Seek(44100 * 10, SeekOrigin.Begin); // Skip 10 seconds at 44.1kHz

// Rewind to beginning
reader.Rewind();

// Close when done
reader.Close();
```

### Chunked Writing

```csharp
using CrossAudio.WavWriter;

// Open for chunked writing
var writer = new WavWriter();
writer.Open("output.wav", sampleRate: 44100, bitsPerSample: 16, channels: 1);

// Write audio data in chunks
var chunk1 = new byte[] { /* PCM data */ };
var chunk2 = new byte[] { /* more PCM data */ };

writer.Write(chunk1);
writer.Write(chunk2);

// Close to finalize the file
writer.Close();
```

## Supported Formats

- **Audio Formats**: Linear PCM (integer)
- **Bit Depths**: 8-bit, 16-bit, 24-bit, 32-bit
- **Channels**: Mono, Stereo, and multi-channel
- **Metadata**: LIST INFO chunks, sampler info, cue points

## Architecture

This library is part of the larger CrossAudio ecosystem, which aims to provide a comprehensive audio solution for .NET applications. The WAV codec is implemented as a pure C# library with no native dependencies, ensuring cross-platform compatibility and easy deployment.

### Key Components

- **WavReader**: Handles WAV file reading and streaming operations
- **WavWriter**: Manages WAV file creation and writing
- **Metadata**: Rich metadata support including artist, title, comments, and technical information
- **SamplerInfo**: Support for sampler chunks with loop points and MIDI information
- **Cue Points**: Precise timing markers within audio files
